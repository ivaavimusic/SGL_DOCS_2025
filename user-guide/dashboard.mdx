# Dashboard Overview

The x402 Studio Dashboard is your central hub for managing endpoints, viewing payments, and analyzing usage.

## Navigation

| Section | Description |
| :--- | :--- |
| **Overview** | Quick stats and recent activity |
| **Endpoints** | Create and manage paid endpoints |
| **Products** | Bundle endpoints into products |
| **Payment Links** | Generate shareable payment pages |
| **Invoices** | Create and track invoices |
| **Components** | Embeddable payment widgets |
| **Settings** | Account and wallet configuration |

## Overview Page

Your dashboard home shows:
*   **Total Revenue:** All-time earnings across all endpoints
*   **Active Endpoints:** Number of endpoints currently live
*   **Recent Payments:** Latest successful transactions
*   **Usage Chart:** API calls over time

## Settings

Manage your account:
*   **Connected Wallets** – Add or remove payment addresses
*   **Payout Preferences** – Default chain for new endpoints
*   **API Keys** – Generate keys for programmatic access
*   **Notifications** – Email alerts for payments
