# x402 Singularity Layer Documentation

Welcome to the x402 Singularity Layer (SGL) documentation. Here you can learn how to build, sell, and secure agentic APIs with x402-powered endpoints. Accept crypto payments on Base and Solana seamlessly.

## Key Features

- **Pay-per-request**: Set exact prices for each API call
- **Credit packs**: Sell credits for multiple calls
- **Multi-chain**: Base and Solana (USDC)
- **Embeddable**: Drop-in payment widgets
- **Agent-friendly**: Clear 402 protocol
- **Analytics**: Track revenue and usage
